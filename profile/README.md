# MESA

**Multidisciplinary Environment for Scientific Advancement**

<img src="MESA-logo-light-1024.png" alt="MESA — stacked-diamond mark" width="140" align="right" />

An open source, agentic AI–driven platform that securely connects publicly funded research data via curated and indexed metadata — reducing time to discovery.

Scientists spend up to **80% of their time** managing data instead of doing science. MESA's AI agents read every dataset as it lands, describe it with standardized, community-curated metadata, and link it to related work across disciplines — collapsing data preparation from **weeks to hours**.

## Architecture

MESA integrates five layers into one coherent platform:

| Layer | Technologies |
|---|---|
| 🖥️ **User Interface** | Laptop · Desktop · Kubernetes containers |
| 🤖 **Agentic AI** | MCP · RAG · Agents · Reasoning · Sandboxing |
| 🗂️ **Data Management** | Data Lakehouse (DuckLake) · Data Mesh (iRODS) |
| ⚙️ **Compute** | CyVerse · TACC · ACCESS-CI · NAIRR |
| 💾 **Object Storage** | S3 · OSN · VAST · Lustre |
